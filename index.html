Challenge Description

Welcome to the Web Developer Challenge! This course is designed to put your skills to the test, with real-world scenarios and projects that will prepare you for a career in web development.

Your Task

This week, your challenge is an on-the-job ticket, which means you will start with a base of starter code that you need to modify. Your goal is to refactor the code to make the website more accessible.

Grading Criteria

Your submission will be graded based on several criteria, including technical accuracy, code organization, and application quality.

Submission Review

After submitting your project, it will be reviewed by the instructional staff. Be sure to submit both the URL of your deployed application and the URL of your GitHub repository.
